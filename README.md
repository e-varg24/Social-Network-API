# Social-Network-API

## Description

This is a Api backend database for a social network. With mongoose,express and the command line users are able to creat,update and delete. You can use insonmia or mongodb compass.

## Installation

- Insomnia
- Mongodb
- Npm install

## Usage

- clone reposistory
- npm run start
